# WP Events Photo Gallery
A WooCommerce plugin for photographers. 

===========================================
DESCRIPTION
==========================================

// Front-End

The administrator can upload photos to the WordPress Media Library and assign them to users.

Users can view the photos assigned to them from their My Account environment. They can then select which photos they want to add to their own albums (wish lists).

Finally, they can select an album and convert it into either a photo album or a mug and proceed to WooCommerce checkout to purchase.

// Back-End

The administrator has the ability to create products that are Events Gallery type and give them the property of a mug, t-shirt, or album so that the user can choose what to match their photos with and purchase the product at WooCommerce checkout.


===========================================
INSTALLATION
==========================================

1. Install ACF Plugin.
2. Upload the file in .zip file from back-end and activate it.
